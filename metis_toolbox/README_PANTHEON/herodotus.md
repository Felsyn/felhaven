# Herodotus

Manage a local archive of Markdown documents. The toolbox's first mutating brain
tool; deliberately has no delete.

- **UI:** none — Pythia-only
- **Brain tools:** `list_documents`, `search_documents`, `read_document`,
  `write_document`, `edit_document`
- **Files:** `tools/herodotus.py`
- **Test:** `python -X utf8 -m unittest tests.test_herodotus`

Details: see the docstring in `tools/herodotus.py`.
