# Morpheus

Play audio from YouTube — search, play, pause, and skip tracks. No video, ever.

- **UI:** Vox Array card → MORPHEUS tab
- **Brain tools:** `play_music`, `resume_music`
- **Files:** `tools/morpheus.py`, `panels/morpheus_panel.py`
- **Test:** `python -X utf8 -m unittest tests.test_morpheus_playlists tests.test_morpheus_panel_smoke`

Details: see the docstring in `tools/morpheus.py`.
