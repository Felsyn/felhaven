# Calliope

Turn a string of text into speech and play it. Handed Pythia's answer text; it
decides nothing about what to speak.

- **UI:** Home view — the speak-aloud control under each answer, and the
  narration lamp in the Hestia control row
- **Brain tool:** none
- **Files:** `calliope.py`, `panels/narrator_panel.py`, `config/calliope_config.json`
- **Test:** `python -X utf8 -m unittest tests.test_calliope`

Details: see the docstring in `calliope.py`.
