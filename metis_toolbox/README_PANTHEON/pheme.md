# Pheme

Aggregate headlines from the configured RSS/Atom news feeds.

- **UI:** Scriptorium view (one tab per feed)
- **Brain tool:** `get_news_stories`
- **Files:** `tools/pheme.py`, `panels/pheme_panel.py`, `config/pheme_rumormill.json`
- **Test:** `python -X utf8 -m unittest tests.test_tool_handles`

Details: see the docstring in `tools/pheme.py`.
