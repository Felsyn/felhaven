# Cerberus

Guard the toolbox's secrets behind one PIN — the sole encryption authority.
Vault, Custody, and Ledger are its three facets.

- **UI:** Moderati card → CERBERUS tab
- **Brain tool:** none — deliberately out of LLM scope
- **Files:** `cerberus.py`, `panels/cerberus_panel.py`, `config/cerberus_manifest.json`
- **Test:** `python -X utf8 -m unittest tests.test_cerberus tests.test_cerberus_panel_smoke`

Details: see the docstring in `cerberus.py`.
