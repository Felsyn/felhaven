# Harmonia

Own the one audio output device. Calliope and Orpheus hand it PCM; nothing else
calls `sounddevice`.

- **UI:** none — infrastructure, like Kairos
- **Brain tool:** none
- **Files:** `harmonia.py`
- **Test:** `python -X utf8 -m unittest tests.test_harmonia`

Details: see the docstring in `harmonia.py`.
