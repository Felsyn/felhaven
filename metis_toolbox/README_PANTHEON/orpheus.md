# Orpheus

Play back one audio file from local_audio/ — the folder Echo writes to. Play and
stop only.

- **UI:** Vox Array card → ORPHEUS tab
- **Brain tool:** none
- **Files:** `tools/orpheus.py`, `panels/orpheus_panel.py`
- **Test:** `python -X utf8 -m unittest tests.test_orpheus tests.test_orpheus_panel_smoke`

Details: see the docstring in `tools/orpheus.py`.
