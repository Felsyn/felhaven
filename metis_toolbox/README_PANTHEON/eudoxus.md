# Eudoxus

Convert between units of length, volume, weight, temperature, and time.

- **UI:** Cogitator card → EUDOXUS tab
- **Brain tool:** `convert_unit`
- **Files:** `tools/eudoxus.py`, `panels/eudoxus_panel.py`
- **Test:** `python -X utf8 -m unittest tests.test_tool_handles`

Details: see the docstring in `tools/eudoxus.py`.
